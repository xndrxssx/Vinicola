Apresentação

Esse repositório é o resultado do meu primeiro projeto acadêmico (e pessoal) com HTML, CSS e Javascript.

BIA - UNIVASF

O projeto foi desenvolvido sob a orientação do Prof. Dr. Daniel Costa e fomentado pelo programa de Bolsa de Iniciação Acadêmica da Universidade Federal do Vale do São Francisco.

Projeto de extensão

O objetivo principal do projeto foi agregar valor aos vinhos orgânicos com uso de tecnologias digitais melhorando a experiência do consumidor final.

Resultado

Desenvolver protótipo de plataforma web para solução tecnológica imersiva; integrar o conteúdo digital, a plataforma web, e a rastreabilidade. Por questões de políticas de confidencialidade, não tenho a autorização para divulgar diretamente o nome da empresa para a qual foi desenvolvido o projeto.
